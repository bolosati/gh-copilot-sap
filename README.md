
# GitHub Copilot para los amantes de SAP 💙

- Docker 🐋 instalado en tu máquina.

- Clona este repo 👩‍💻
- O también puedes ejecutarlo en GitHub Codespaces 🐈‍⬛ [Echa un vistazo a este vídeo](https://www.youtube.com/watch?v=0qKG37C8sb8)
